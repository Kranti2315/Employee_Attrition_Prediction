# Employee Attrition Prediction System

## Overview

The Employee Attrition Prediction System is a Machine Learning-based web application developed to predict whether an employee is likely to stay with or leave an organization. The application analyzes employee-related information and assists organizations in identifying employees who may be at risk of attrition, enabling timely retention strategies and informed decision-making.

The application is developed using Python, Scikit-learn, and Streamlit.

---

## Problem Statement

Employee attrition is a significant challenge for organizations as it directly impacts productivity, recruitment costs, and overall business performance. Predicting employee attrition enables Human Resource departments to identify potential risks and implement effective employee retention strategies.

---

## Objectives

- Predict employee attrition using Machine Learning techniques.
- Compare the performance of multiple supervised classification algorithms.
- Select the best-performing model based on evaluation metrics.
- Deploy the trained model as an interactive web application using Streamlit.
- Provide an intuitive interface for predicting employee attrition.

## Machine Learning Models

The following supervised classification algorithms were trained and evaluated:

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Gaussian Naive Bayes
- Gradient Boosting Classifier
- XGBoost Classifier (Optional)

The final model was selected based on the following evaluation metrics:

- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC Score

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd Employee-Attrition-Prediction
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run app.py
```

## Model Output

The application predicts one of the following outcomes:

- Stayed
- Left

The prediction is accompanied by confidence scores generated by the trained classification model.

---

## Application Workflow

1. Load the trained Machine Learning model.
2. Collect employee information through the Streamlit interface.
3. Preprocess the input data using the saved preprocessing pipeline.
4. Predict employee attrition.
5. Display prediction results and confidence scores.
6. Allow users to download the entered employee details.
